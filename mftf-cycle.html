Program Overview

Music for the Future

Justice settings · Annual operating cycle
What this is
A year-round creative music program for incarcerated individuals. Each cohort moves through a seasonal arc: initial residency, remote coursework, a full production, and a premiere — culminating in a celebration streamed live from Lincoln Center.
Current partners
  • Five Keys (California — Chowchilla)
  • Expansion underway: Connecticut, Nevada
Core team (program)
  • Dana — artistic lead, co-writes the musical
  • Michael — co-writer, present through production
  • SFCM faculty — remote course instruction
  • TAs embedded in each residency
